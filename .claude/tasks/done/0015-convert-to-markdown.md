# Problem to solve

HTML files and images are clean and ready to be converted to Markdown. All data are stored locally in `assets/html` and `assets/img`. The links between the HTML files are correct and the links in the HTML to the images are correct.

The objective of this task is to convert the content of the HTML files into Markdown to extract the real content.

For each HTML file in `assets/html`, convert it into `assets/md` with the constraints below:

- The links to other pages must be adapted to the Markdown converted page.
- The links to the images must be adapted and must be working in the Markdown converted page.
- The header levels must be kept between the HTML and the Markdown.
